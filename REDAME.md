# 🌍 AI-Powered Country Clustering Dashboard

An interactive Machine Learning dashboard that analyzes and clusters countries based on socio-economic indicators using Unsupervised Learning techniques.

This project applies data preprocessing, dimensionality reduction, and clustering algorithms to identify patterns and similarities between countries. The dashboard provides rich visualizations and interactive analytics using Streamlit and Plotly.

---

## 🚀 Live Demo

[Open Dashboard](https://clustering-countries-ai.streamlit.app)

---

# 📌 Features

- 📊 Interactive data visualizations
- 🌍 Country clustering analysis
- 🤖 K-Means clustering implementation
- 📉 PCA dimensionality reduction
- 📈 Dynamic charts using Plotly
- 🎨 Modern glassmorphism UI
- ⚡ Real-time dashboard interaction

# 🧠 Machine Learning Techniques Used

- Data Cleaning & Preprocessing
- Feature Engineering
- Standard Scaling
- Principal Component Analysis (PCA)
- K-Means Clustering
- Exploratory Data Analysis (EDA)
